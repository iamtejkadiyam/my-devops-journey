## Search and Text Processing

Used to search files, logs, and directories efficiently.

### grep
Searches text using patterns.

Example:
grep "error" app.log

---

### grep -r
Searches recursively in directories.

Example:
grep -r "PORT" /etc

---

### find
Searches files and directories.

Example:
find /var/log -name "*.log"

---

### DevOps Usage
- Searching logs for errors
- Finding configuration files
- Debugging production issues
