## Bash Basics for DevOps

Bash scripting is used for automation and operational tasks.

### Variables
Used to store values.

Example:
name="devops"

---

### Conditionals
Used for decision making.

Example:
if [ "$name" = "devops" ]; then
  echo "DevOps user"
fi

---

### Loops
Used for repetitive tasks.

Example:
for i in 1 2 3
do
  echo $i
done

---

### Scripts
Combines commands into executable files.

Example:
#!/bin/bash
echo "Hello DevOps"

---

### DevOps Usage
- Automation scripts
- CI/CD helper scripts
- Server setup tasks
