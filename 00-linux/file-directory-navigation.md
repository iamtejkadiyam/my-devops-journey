## File and Directory Navigation

Used to move around the Linux filesystem and understand directory structure.

### pwd
Shows the current working directory.
Used to confirm location before executing commands.

Example:
pwd

---

### ls
Lists files and directories.

Example:
ls

---

### ls -l
Displays detailed file information including permissions, owner, and size.
Used to debug permission issues.

Example:
ls -l

---

### cd
Changes the current directory.

Examples:
cd /var/log
cd /etc/nginx

-----

### DevOps Usage
- Navigating servers
- Locating logs and config files
- Verifying deployment paths
