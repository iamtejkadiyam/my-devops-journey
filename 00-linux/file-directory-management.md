## File and Directory Management

Used to create, delete, copy, and move files and directories.

### mkdir
Creates a new directory.

Example:
mkdir app_logs

---

### rmdir
Removes an empty directory.

Example:
rmdir temp_folder

---

### rm
Deletes files or directories.

Example:
rm file.txt
rm -r old_folder

---

### cp
Copies files or directories.

Example:
cp config.yml backup.yml

---

### mv
Moves or renames files.

Example:
mv app.log app_old.log

---

### DevOps Usage
- Managing application files
- Cleaning logs
- Organizing build artifacts
