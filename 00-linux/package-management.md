## Package Management

Package managers are used to install, update, and remove software on Linux systems.

### apt update
Updates the local package index.

Example:
apt update

---

### apt install
Installs a package.

Example:
apt install nginx

---

### apt remove
Removes a package.

Example:
apt remove nginx

---

### apt search
Searches for available packages.

Example:
apt search docker

---

### dpkg -l
Lists installed packages.

Example:
dpkg -l

---

### DevOps Usage
- Installing Docker, Kubernetes tools, CI agents
- Managing server dependencies
- Preparing cloud virtual machines
