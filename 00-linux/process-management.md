## Process Management

Used to monitor and control running processes.

### ps
Shows running processes.

Example:
ps -u username

---

### top
Displays live system processes and resource usage.

Example:
top

---

### kill
Stops a process using PID.

Example:
kill 1234

---

### kill -9
Force kills a process.

Example:
kill -9 1234

---

### pkill
Kills processes by name or user.

Example:
pkill nginx

---

### DevOps Usage
- Stopping stuck applications
- Managing server resources
- Debugging high CPU/memory usage
